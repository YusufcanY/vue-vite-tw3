### This boilerplate have:

- Vue 3
- Vue Router Next (v4)
- Vite 4
  - SVG Loader
  - Eslint Plugin
- Tailwindcss 3
- PostCSS 8
- Eslint

for install the node_modules

```
yarn
```

for run development

```
yarn dev
```

for run build and start on production

```
yarn test
```

for format all files with prettier

```
yarn format
```
